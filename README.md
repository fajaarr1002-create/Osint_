## ⛩ Description

OSINTJAR is a lightweight Open Source Intelligence toolkit built with Python.

Designed for fast information gathering directly from the terminal with a simple interface and minimal dependencies.

The tool allows users to analyze publicly available information related to phone numbers, IP addresses, and usernames across multiple online platforms.

## ⚙ Installation

### Termux

```bash
pkg update && pkg upgrade -y

pkg install git python -y

git clone https://github.com/fajaarr1002-create/Osint_.git

cd Osint_

python osint.py
```

### Manual Installation

```bash
git clone https://github.com/fajaarr1002-create/Osint_.git
```
